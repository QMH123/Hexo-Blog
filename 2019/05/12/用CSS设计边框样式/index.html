用CSS设计边框样式

2019/05/12

CSS为元素边框定义了众多样式

1.边框样式-border-style

虚线框
dotted 点
dashed 虚线

实线框
solid 实线框
double 双线框
groove 立体凹槽
ridge 立体凸槽
inset 立体凸边
outset 立体凹边

2.边框颜色和宽度

boder-color属性可以定义边框的颜色

boder-width属性可以定义边框的粗细

定义单边框的样式

border-top/right/left/bottom-color可以定义单边框的颜色
border-top/right/left/bottom-width可以定义单边框的宽度

同一个边框也可以配合使用不同的样式

border-style: solid dashed dotted double ;
表示上边实线，右边虚线，底边点线，左边双线

border-width : 10px 20px 30px 40px ;
顶边10px 右边20px 下边30px 左边 40px

border-color: red blue green yellow;
上边红色 右边蓝色 下边绿色 左边黄色
记忆顺序：上右下左

定义圆角边框
使用CSS中border-radius属性，使用他可以设计圆角样式
border-radius属性可包含两个参数值
border-radius:10px;

border-radius: 50%;

为边框添加阴影效果
使用box-shadow属性为边框添加阴影效果

标准用法

box-shadow: 10px 20px 20px #888888;

三个参数分别为阴影水平偏移、垂直偏移，以及阴影大小（及阴影模糊程度）
以及阴影颜色
