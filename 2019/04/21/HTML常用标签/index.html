HTML常用标签

2019/04/21

学习html知识后，发现自己对许多html标签的使用还不是很熟悉，于是决定写一篇博客来汇总最近经常用到的标签以及一些具有相同作用标签之间的区别与联系。

< html>< /html>: 标识html文档的起始和终止

< head>< /head>:标识html文档的头部区域

< body>< /body>:标识html文档的主体区域

< p>< /p>: 段落标签 标识段落文本

< br/> 换行

< hr/>水平线

< h1>标签和< title>标签的区别
h1标签： 写在文章正文的标题部分，是展示给用户看的。

title标签： 写在网页的head部分，是展示给搜索引擎看的，搜索引擎的搜索结果中展示的标题就是这个title标签里的内容。title标签不会对用户展示。< hi>< /hi>: 标识标题文本，i=1、2、3 、4、5、6，分别表示一级到六级标题。

< blockquote>< /blockquote>和< q>< /q>的区别与联系
第一： q标签为行内标签，blockquote标签为块标签。

第二： q标签一般用于引用比较短的文字，blockquote一般用在一大段的引用，并且blockquote标签会自己缩进一个tab键的距离。

< meta>标签
用在< head>< /head>标签之间
character属性: 规定HTML文档的字符编码 < meta charset=”character_set”>

name 属性：
< meta name=”KEYWords” contect=””>向搜索引擎说明你的网页的关键词；
< meta name=”DEscription” contect=””>告诉搜索引擎你的站点的主要内容；
< meta name=”Author” contect=”你的姓名”>告诉搜索引擎你的站点的制作的作者；

< link> 标签:
定义文档与外部资源的关系。最常见的用途是链接样式表。
href属性: 定义被链接文档的位置rel属性：定义当前文档与被链接文档之间的关系。
type属性： 规定被链接文档的类型
常见用法：
< head> < link rel=”stylesheet” type=”text/css” href=”theme.css”> < /head>

< style> 标签
定义 HTML 文档的样式信息。在 < style> 元素中，您可以规定在浏览器中如何呈现 HTML 文档。（但人们通常使用css样式表

< div>标签：
< div> 标签可以把文档分割为独立的、不同的部分。它可以用作严格的组织工具，并且不使用任何格式与其关联。如果用 id 或 class 来标记 < div>，那么该标签的作用会变得更加有效。

< div> 是一个块级元素。这意味着它的内容自动地开始一个新行。

< div>、< article>、< section>标签的区别和联系
针对一个块内容做样式化，三者并无区别。
div、section、article，语义是从无到有，逐渐增强的。div无任何语义，仅仅用作样式化或者脚本化的标签。对于一段主题性的内容，则就适用section，而假如这段内容可以脱离上下文，作为完整的独立存在的一段内容，则就适用 article。原则上来说，能使用article的时候，也是可以使用section的，但是实际上，假如使用article更合适，那么就不要使用section。

< span>< /span>标签：
提供一种途径，令你可以对文档中的一部分进行引用。对文档中的行内元素进行组合这样，你便可以引用部分文本，并对它们设置样式，或用JavaScript对它们进行处理。

< nav>< /nav>标签:
常配合< a>标签来制作导航栏

有序列表< ol>< /ol> < li>< /li>
次序是排序的重要依据，有序列表的特点是所输入的内容前有数字标记。

无序列表< ul>< /ul> < li>< /li>
出现的顺序无所谓，无序列表的特点是所输入的内容前有加粗效果的小圆点作为标记。

自定义列表（dl-dt-dd）
在自定义列表中，在dt和dd中有了一个缩进。在有序和无序列表要实现此结构就要用到列表的嵌套

表格< table>< /table>
表格行< tr>< /tr>
表格单元格< td>< /td>
表头< th>< /th>
标题< caption>< /caption>


< select> < /select> 与< option>< /option>
结合使用下拉菜单及选项当select元素定义size属性后，将会由下拉菜单的形式转换为列表框形式。

option常用属性：
value=””:当option没有value属性时，往后台传递的是< option>< /option>标签中的文字；当option有value属性时，往后台传递的是value属性的值。

title=””:鼠标指上后显示的文字。

selected=”selected”:默认选中。

< textarea>文本框
可以通过 cols 和 rows 属性来规定 textarea 的尺寸大小，不过更好的办法是使用 CSS 的 height 和 width 属性。

placeholder属性 ：在文本框内给予提示

readonly=”readonly” 设置为只读模式，不允许编辑。

< img>< /img>标签
src属性表示图片引用路径。
常见路径的写法：
①相对路径：
Ⅰ -当图片在当前文件下一层时：文件夹名/图片名 例如：img/abc.jpg

Ⅱ-当图片与当前文件在同一层时：图片名 例如：src=”abc.jpg”Ⅲ-当图片在当前文件上一层时：../图片名 例如：src=”../abc.jpg”使用相对路径时，图片最外层只能放到网站根目录（图片必须要在项目文件夹中）

②绝对路径：写法file：///E:/aaa.png 但是，严禁使用

③网络连接：直接使用图片的网络地址，但由于图片在别的服务器，不可控，故不建议使用

title属性:图片的标题。当鼠标指上时，显示的提示文字。

alt属性 :当图片无法加载时显示的文字。

width/height属性: 图片的尺寸，相当于CSS中的style=”width:”

align属性: 图片周围的文字，相对于图片的排列方式。
可选值：top/center/bottm em（强调）：浏览器显示为倾斜。

【strong/em标签的区别】
em和strong都表示强调，strong和em标签均可多层嵌套，表示强调程度的递增。
