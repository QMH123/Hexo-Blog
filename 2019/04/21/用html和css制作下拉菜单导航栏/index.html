使用HTML和CSS制作下拉导航栏


平时浏览网页时，我们常常能在各种网页的导航栏部分见到如效果图所示的导航栏，通过上网搜集资料并浏览别人写的博客，在此总结一下方法和其中的一些注意事项。
导航栏总体架构
使用 < nav > 标签，其中嵌套< ul >无序列表来构建总体框架生成一二三级菜单。

<nav>       
<ul>           
<li><a href="#">导航一</a>                
    <ul>                   
     <li><a href="#">二级导航</a></li>                   
     <li><a href="#">二级导航</a></li>                   
     <li><a href="#">二级导航</a></li>                   
     <li><a href="#">二级导航</a></li>                
     </ul>            
</li>            
<li><a href="#">导航二</a>                
  <ul>                    
  <li><a href="#">二级导航</a></li>                    
  <li><a href="#">二级导航</a></li>                    
  <li><a href="#">二级导航</a></li>                    
  <li><a href="#">二级导航</a></li>                
  </ul>           
</li>            
<li><a href="#">导航三</a>                
<ul>                    
  <li><a href="#">二级导航</a></li>                    
  <li><a href="#">二级导航</a>                        
    <ul>
    <a href="#">三级导航</a>                            
    <a href="#">三级导航</a>                            
    <a href="#">三级导航</a>                            
    <a href="#">三级导航</a>                       
    </ul>                    
  </li>                    
  <li><a href="#">二级导航</a></li>                    
  <li><a href="#">二级导航</a></li>                
</ul>                  
</ul>   
</nav>

这样生成了一个不含任何装饰的导航栏

使用html将基本框架搭建完成后，我们使用css样式来达到我们想要的效果。首先我们给每一个< ul >和< li >设计样式。

在这里我们先了解一些css属性
1.text-aling:
他可以有left right center 三个值，分别代表设置块级元素内文本的水平对齐方式，分别代表左对齐，中间对齐，右对齐。

2.display:
display:none
将元素设置为none的时候既不会占据空间，也无法显示，相当于该元素不存在。

display:inline
将元素作为行内元素显示，它可以让行内显示为块的元素，变为行内显示，例如
< div> DIV1 < /div>
< div> DIV2 < /div>
这里DIV1和DIV2分别占一行，但是你给他们加上属性后变了
< div style= “display:inline “ > DIV1 < /div>
< div style= “display:inline “ > DIV2 < /div>
DIV1和DIV2这时候显示在同一行了

display:block
设置元素为块状元素，并独占一行。

display:inline-table
inline-table得到的是，外面是“内联盒子”，里面是“table盒子”。
得到的是一个可以和文字在一行中显示的表格。

3.position：
position 属性规定元素的定位类型。
absolute值
生成绝对定位的元素，相对于 static 定位以外的第一个父元素进行定位。
元素的位置通过 “left”, “top”, “right” 以及 “bottom” 属性进行规定。
relitive值
生成相对定位的元素，相对于其正常位置进行定位。
因此，”left:20” 会向元素本来正常所在的 LEFT 位置添加 20 像素。

4.伪选择器: hover
选择鼠标指针浮动在其上的元素，并设置其样式

解完这些知识后我们开始调整css样式

nav{    
margin :100px aotu;    
text-align: center;
}

nav ul{    
border-radius: 10px;   
background: linear-gradient(to bottom, #efefef ,#bbbbbb);    
padding:0 20px;    
display: inline-table;    
position: relative;    
box-shadow: 1px 1px 3px #666;    
}

nav ul ul{    
display: none;
}

nav ul li {  float: right;}

nav ul li a{    
display: block;    
padding: 25px 40px;   
 color: #000;    
 text-decoration: none;    
 font-family: "微软雅黑"；
 }

首先用text-aling设置导航栏内文本居中。
然后设计第一级导航栏的样式

其中border-radius属性可以给块状元素添加圆角边框来达到美化效果

使用box-shadow属性给块状元素的附近添加阴影效果

过padding 来设置导航栏的内外边距

设置第一级导航栏中< a >标签的长宽，文本的字体，颜色，并将其作为一个块状区域显示。

将第二级导航栏的display设置为none，将其隐藏。然后我们打开浏览器显示效果。

我们发现第一级导航并没有在同一行内显示，这是因为在css中我们将他们都显示为了块状元素，他们之间会自动换行，解决方法是将第一级导航栏的每一项设置为浮动元素，使其在一行内显示。

经过这番操作后，导航栏总体框架已经形成。

我们可以看出此时导航栏的外形与最终形成的版本外形上已无任何区别，只是当鼠标悬停在导航栏上时不会显示任何的效果。

现在我们通过hover来添加当鼠标悬停在导航栏上时的动态效果

nav ul li:hover {    
background: linear-gradient(to bottom,#4f5964,#5f6975);
}
nav ul li:hover a{    color:#FFF;}

设置鼠标悬停在导航栏上时，被悬停的区域背景变为深色，其中的文本变为白色，给用户此项被选中的交互信息。

然后我们把被选中的第一级导航栏中的< ul>的display设置为block,使其作为一个块状区域重新显示。调整完这一步后，我们重新把鼠标悬停在导航上。


我们发现还是没有得到我们想要的效果，第二级导航栏仍然在一行内显示，这是因为我们还未对第二级导航栏的css进行设置，所以他自动继承了他的父元素的属性，成为了浮动元素，现在我们开始对第二级导航栏的css样式进行设置。

nav ul ul {    
background: #5f6975;    
border-radius: 0;    
position: absolute;    
top: 100%;    
padding: 0;}

nav ul ul li{    
float: none;    
border-top: 1px solid #6b727c;   
 border-bottom: 1px solid #575f6a;}

我们将第二级导航栏的position值设置为absolute,并将top设置为100%，这表明了相对于第二级导航栏的父级元素，他会在其正下方出现，我们将border-radius和padding重新设置为0来消除来自父级元素的继承。

然后将第二级导航栏中的每一项的float设置为none,这样列表的每一项就都会在不同行显示，为了美观我们给列表的每一项的上下边框设置颜色便于区分。现在再用浏览器打开网页，二级菜单的所有效果就初步实现。但第三级导航栏自动继承了第二级导航栏的css属性，显示在其下方。

我们希望第三级导航栏能在我们所选中的第二级导航栏的右边显示，于是我们再次设置其css样式。

nav ul ul li a:hover{    background: #4b545f;}

nav ul ul ul {    
 width: 100%;   
 position: absolute;    
 left: 100%;    
 top: 25%;}

我们设置第三级导航栏相对其父级元素的位置为其右方，但在设置top的值时需注意，在本例中我们的第三级菜单在第二项，其中父级菜单有4项，所有为了让其显示在他正右方，我们将top设置为1/4，即25%。
由此我们可推出公式，若其父级菜单有n项，而他处于其父级菜单的第m项中，为了能将其显示在他父级元素的正右方，我们可以将top设置为（m-1)/n。

此时我们的导航栏便全部制作完成。
